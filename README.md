# QrCodeGeneration

This Python script generates a QR code containing a specified URL using the `qrcode` library.

## Usage

1. Install the `qrcode` library using pip:

   ```bash
   pip install qrcode[pil]
2. Run the script `QRCode.py` to generate the QR code.

3. The generated QR code will be saved as `qrcode.png` in the current directory.
